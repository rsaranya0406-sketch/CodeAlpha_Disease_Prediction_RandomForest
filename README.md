# ❤️ Heart Disease Prediction using Random Forest

## 📌 Project Overview

This project predicts whether a patient is likely to have heart disease using the Random Forest Classification algorithm.

The model is trained using a heart disease dataset and evaluates patient health parameters to classify the presence of heart disease.

This project was developed as part of the **CodeAlpha Machine Learning Internship**.

---

## 🎯 Objectives

- Predict heart disease using Machine Learning.
- Train a Random Forest Classifier.
- Evaluate model performance.
- Visualize Confusion Matrix.
- Plot ROC Curve.
- Display Feature Importance.
- Perform sample predictions.

## 📊 Dataset

The dataset contains patient medical information including:

- Age
- Sex
- Chest Pain Type
- Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- ECG Results
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Slope
- Number of Major Vessels
- Thalassemia
- Target (Heart Disease)

---

## ⚙️ Machine Learning Algorithm

Random Forest Classifier

Parameters used:

- n_estimators = 100
- random_state = 42

## ▶️ How to Run

Clone the repository

```bash
git clone https://github.com/YourUsername/CodeAlpha_Disease_Prediction_RandomForest.git
```

Move into project

```bash
cd CodeAlpha_Disease_Prediction_RandomForest
```

Install required packages

```bash
pip install -r requirements.txt
```

Run

```bash
python disease_prediction.py
```

or

Open

```
disease_prediction.ipynb
```

in Jupyter Notebook.
